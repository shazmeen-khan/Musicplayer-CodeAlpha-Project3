# 🎵 Music Player

A modern and responsive Music Player built using **HTML, CSS, and JavaScript**. This project provides an interactive music listening experience with essential playback controls, dynamic song information, and playlist support.

---

## 🚀 Features

- 🎧 Play & Pause Music
- ⏭ Next & Previous Song
- 🎵 Dynamic Song Title & Artist
- ⏱ Song Duration Display
- 📊 Interactive Progress Bar
- 🔊 Volume Control
- 📃 Playlist Support
- 🔁 Autoplay Next Song
- 📱 Fully Responsive Design
- ✨ Clean & Modern User Interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
